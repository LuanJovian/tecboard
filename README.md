# Website - Tecboard App
## Sobre
- Página comercial de promoção do aplicativo **Tecboard** de monitoramento online.   
- Projeto baseado nas aulas dos cursos _"HTML e CSS: ambiente, estrutura e estilo"_ e _"HTML e CSS: formulários, SEO e acessibilidade"_, realizados na plataforma **Alura**.
## Objetivos didáticos
A construção deste repositório teve como finalidade desenvolver e praticar conhecimentos adquirdos nos cursos mencionados acima, como:
- Estruturação de HTML semântico,
- Estilização CSS baseada em design Figma,
- Técnicas de otimização para mecanismos de busca (SEO),
- Publicação de projeto no GitHub Pages.
